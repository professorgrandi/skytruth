## ğŸ‡¬ğŸ‡§ English

### Project Overview
SkyTruth is an open-source atmospheric monitoring system that combines visual sky observation with radio signal interception to document aerial activity. The project aims to increase transparency in airspace monitoring through automated data collection and analysis.

### Key Features
- **Automated Timelapse Capture** - Continuous sky monitoring with timestamped imagery
- **ADS-B Signal Interception** - Real-time aircraft tracking using software-defined radio
- **Multi-source Correlation** - Cross-reference data with OpenSky Network and Flightradar24
- **Anomaly Detection** - Automated identification of unusual aerial activity
- **Privacy-First Design** - Optional Tor network deployment for anonymous reporting

### Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Data Sources**: ADS-B, OpenSky API, Flightradar24
- **Visualization**: Custom timelapse generation, interactive maps
- **Deployment**: GitHub Pages, Tor onion service optional

### Purpose
This project serves researchers, aviation enthusiasts, and transparency advocates interested in independent airspace monitoring and atmospheric data collection.
